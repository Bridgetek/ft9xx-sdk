Detailed Description

USB CDC class USB definitions.

This contains USB definitions of constants and structures which are used in USB CDC implementations.

Definitions for USB CDC Devices based on USB Class Definitions for Communications Devices Specification Revision 1.1 from http://www.usb.org/developers/docs/
