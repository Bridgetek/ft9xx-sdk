unsigned char USB_UVC_VS_CSStillImageFrameDescriptor::bEndpointAddress

If method 3 of still image capture is used, this contains the address of the bulk endpoint used for still image capture. The address is encoded as follows: D7: Direction. (set to 1 = IN endpoint) D6..4: Reserved, reset to zero D3..0: The endpoint number, determined by the designer If method 2 of still image capture is used, this field shall be set to zero.
